Lab 6: Expense Tracker (Firebase)

พัฒนาระบบบันทึกข้อมูลรายรับ-รายจ่ายโดยใช้ Ionic Vue ร่วมกับ Firebase Firestore รองรับการทำงานแบบ CRUD:

  • Calculation: คำนวณยอดรวมรายรับ-รายจ่ายแบบอัตโนมัติ
  • Cloud Storage: ข้อมูลไม่หายเมื่อปิดแอปหรือเปลี่ยนเครื่อง
