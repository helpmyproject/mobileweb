รายละเอียดผลงาน Lab 6: Expense Tracker

พัฒนาระบบบันทึกข้อมูลรายรับ-รายจ่ายโดยใช้ Ionic Vue ร่วมกับ Firebase Firestore รองรับการทำงานแบบ CRUD เต็มรูปแบบ:

  • Create: บันทึกข้อมูลผ่านหน้า Tab 2
  • Read: แสดงผล Realtime ผ่าน onSnapshot ในหน้า Tab 1
  • Update/Delete: จัดการข้อมูลผ่านหน้า EditExpense
