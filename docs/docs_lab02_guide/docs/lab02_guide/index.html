นายณัฐดนัย ชาวไทย

ยินดีต้อนรับสู่ ม.ขอนแก่น

หน้านี้รวบรวมสถานที่น่าสนใจภายในมหาวิทยาลัยขอนแก่น (KKU) ทั้งตึกเรียน หอสมุด ศูนย์อาหาร และพื้นที่กิจกรรม เพื่อช่วยนักศึกษาใหม่และผู้เยี่ยมชมเดินทางง่ายขึ้น

หอสมุดกลาง (Central Library)

หอสมุดกลางของมหาวิทยาลัยขอนแก่น เป็นศูนย์รวมหนังสือ วารสาร และที่นั่งอ่านสำหรับนักศึกษา มีพื้นที่สงบและบริการยืมคืนทรัพยากรการเรียนรู้

ศูนย์อาหารกลาง (Complex)

ศูนย์อาหารรวมร้านอาหารหลากหลาย ตั้งอยู่ใจกลางมหาวิทยาลัย เป็นจุดพบปะของนักศึกษาและคณาจารย์ เหมาะสำหรับมื้อกลางวันหรือพักผ่อนระหว่างเรียน

วิทยาลัยการคอมพิวเตอร์ (CP KKU)

ตึกวิทยาลัยการคอมพิวเตอร์เป็นที่ตั้งของภาควิชาต่าง ๆ ห้องปฏิบัติการ และกิจกรรมชมรมด้านเทคโนโลยี เหมาะสำหรับการเรียนและทำโปรเจกต์

สนามกีฬา (Sports Complex)

สนามกีฬาในมหาวิทยาลัยมีทั้งสนามแข่งขันและพื้นที่ออกกำลังกาย เหมาะสำหรับการฝึกซ้อม การจัดการแข่งขัน และกิจกรรมชมรมกีฬาต่าง ๆ
